(constant) data

This is the data that will be loaded to your page. Alernatively you can configure the getData function to fetch from an API but the data has to have the same shape as this The data is a array of objects with the following properties

Properties:
Name Type Description
index number

the user index. Must be unique
picture string

the url to the image
name string

the First and Last name
company string

the company name
status string

should be either Consulting, Live or Data
email string
telephone string
bio string
hobbies Array.<string>

the list of hobbies
tags Array.<string>

the list of tags
missions Array.<object>

the list of missions

Properties
Name Type Description

the mission title
date string

the mission date
